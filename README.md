# fox_and_geese
## курсовая по АиП ##

### игровое приложение "Лиса и гуси"
Цель лисы - срубить как можно больше гусей. <b>Победа лисы</b> фиксируется если осталось только 8 гусей.  
Цель гусей - лишить лису возможности хода. Если гусям это удастся, то это для <b>гусей победа</b>.  
Но вожможна ситуация, когда гуси разбегутся по углам, и тогда лиса не сможет их срубить, а гуси не смогут запереть лису. В этом случае фиксируется <b>ничья</b>.

#### Правила хода #
Лиса одна и она ставится в начале игры на любое свободное поле.  
Неравенство фигур (лиса одна, а гусей - 13) компенсируется разными возможностями.  
Гуси могут перемещаться на соседнее поле по вертикали или горизонтали и это всё.  
Лиса ещё может перемещаться по диагональным направлениям.  
Кроме того, лиса может рубить гусей. Рубка осуществляется шашечным способом, т.е. прыжком через гуся, при условии, что целевое поле свободно. Если после рубки гуся, становится возможна рубка ещё одного гуся, то лиса имеет право опять рубить этим же ходом. Таким образом, рубка в этой игре серийная и лиса за один ход может срубить много гусей.  

#### Анализ игры #
Игра простая, поэтому и стратегические принципы не отличаются сложностью.  
Лисе необходимо удерживаться на пяти центральных полях, обладающих наибольшим количеством степеней свободы. Пять точек образующих крест в центре игрового поля (центральный крест) - это стратегические точки для лисы.  
Важный совет для лисы: т.к. гуси срубить лису не могут, а её цель именно поедание гусей, то лиса должна играть всегда в контакте с массивом гусей.

Для гусей ситуация диаметрально противоположная.  
Задача гусей оттеснить лису с стратегически важных полей центрального креста. Для этого им необходимо двигаться общей массой, чтобы не попасть под рубку. Идея очень простая, гуси должны выдвигаться на поля ещё не занятые лисой и затем укреплять тыл, опять выдвигаться и укрепляться, чтобы не допустить рубки.
